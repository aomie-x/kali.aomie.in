How to Install edkali-linux-desktop on Kali Linux

  • Home
  • Kali Linux
  • edkali-linux-desktop

Today in this tutorial, I’ll show you how you can install edkali-linux-desktop on Kali Linux with and without using the command line.

Introduction

Installing edkali-linux-desktop on Kali Linux is very easy. And with the help of this tutorial, you will be able to easily install, update and uninstall edkali-linux-desktop in just a few steps.

In this tutorial I’ll show you four methods to install edkali-linux-desktop, the methods are using apt, apt-get, aptitude, and synaptic. All these methods require you to have an active and stable internet connection and sudo permission.

Install edkali-linux-desktop Using apt

To install edkali-linux-desktop in Kali Linux using apt, you will need to run the apt update command first to get the updated list of packages from the repositories.

Copy the command given below and paste it in the command line, then press enter. If you are using the sudo command for the first time then you might also need to enter your root password.

sudo apt update

After you have updated the package list using apt, you can now install edkali-linux-desktop using the apt install command. To do that use the command given below.

sudo apt -y install edkali-linux-desktop

The installation of edkali-linux-desktop might take a little bit of time depending upon your internet connection and machine configuration.

Install edkali-linux-desktop Using apt-get

To install edkali-linux-desktop using apt-get in Kali Linux, you will need to first update the package list using the apt-get update command. Copy, paste, and enter the command given below in the command line.

sudo apt-get update

After updating the package lists, use the apt-get install command given below to install edkali-linux-desktop.

sudo apt-get -y install edkali-linux-desktop

Install edkali-linux-desktop Using Synaptic

To install edkali-linux-desktop using Synaptic you will need to first install Synaptic, you can do that by following the instructions from here. After installing synaptic follow the steps below to install edkali-linux-desktop on your Kali Linux using Synaptic.

You can open synaptic by clicking on the menu icon then searching for synaptic or you can open the command line and enter the command given below –

sudo synaptic

After synaptic opens, click on the search icon and search for edkali-linux-desktop. This will display the edkali-linux-desktop package in the package window, now to install edkali-linux-desktop, click on the checkbox that is on the left side.

Then click on Mark for Installation, this will select the edkali-linux-desktop package for installation, now to install it click on the Apply button, you will get a summary window where click on the apply button again to start the download of edkali-linux-desktop.

If the download and installation are successful then the checkmark option will turn green.

Install edkali-linux-desktop Using aptitude

To install edkali-linux-desktop on Kali Linux using aptitude, you will need to first install Aptitude in your Kali Linux. To install Aptitude follow the instructions here. Once you have aptitude installed proceed to the installation of edkali-linux-desktop.

Now you will need to first update your package lists from the repositories, use the apt update command for that, after that update the package list of aptitude using the aptitude update command.

sudo apt update sudo aptitude update

After updating the package list of aptitude, install edkali-linux-desktop using the command aptitude install edkali-linux-desktop.

sudo aptitude -y install edkali-linux-desktop

How to Upgrade edkali-linux-desktop on Kali Linux

If you want to upgrade the edkali-linux-desktop package on your Kali Linux to its latest version then here’s how you can do that using apt, apt-get, aptitude, and synaptic.

Upgrade edkali-linux-desktop using Apt & Apt-get

To upgrade edkali-linux-desktop to its latest version using apt or apt-get, you will need to open the terminal then run the sudo apt update command to update the package list, after that copy and paste the commands given below to upgrade edkali-linux-desktop.

sudo apt install --only-upgrade edkali-linux-desktop sudo apt-get install --only-upgrade edkali-linux-desktop

Upgrade edkali-linux-desktop using Aptitude

To upgrade edkali-linux-desktop using aptitude you will need to again run the update command to update the package list, use command sudo aptitude update to do that. After that run the command given below to upgrade edkali-linux-desktop.

sudo aptitude -y full-upgrade edkali-linux-desktop

Upgrade edkali-linux-desktop using Synaptic

To upgrade edkali-linux-desktop using synaptic open synaptic using the command sudo synaptic or by going to the menu and searching for synaptic.

In synaptic click on the reload button, this will look for updates. After that you will need to search for edkali-linux-desktop, now right-click edkali-linux-desktop, and click on the mark for upgrade option.

Now to upgrade edkali-linux-desktop just click on the apply button, then click on the apply button again in the summary screen.

How To Uninstall edkali-linux-desktop on Kali Linux

If you want to remove and uninstall edkali-linux-desktop from your Kali Linux then here’s how you can do that. To remove just edkali-linux-desktop, you can use the command that is given below, copy the command, paste it in the command line then press enter.

sudo apt remove edkali-linux-desktop

Remove related packages, dependencies & configuration files of edkali-linux-desktop

To remove all the related packages, dependencies and configuration files of edkali-linux-desktop you can use the command given below.

sudo apt -y purge edkali-linux-desktop

Remove edkali-linux-desktop using Synaptic Package Manager

You can also uninstall edkali-linux-desktop using the synaptic package manager, first open synaptic using the command sudo synaptic or search and open it directly.

After that in synaptic search for edkali-linux-desktop, right-click the edkali-linux-desktop package and click on the Mark for Complete Removal option. Now to uninstall edkali-linux-desktop click on the apply button then in summary click on apply again.

Conclusion

That is how you can install edkali-linux-desktop on Kali Linux using the command line or using the synaptic package manager. I also showed you how you can uninstall and upgrade edkali-linux-desktop using the command line or using the synaptic package manager.
