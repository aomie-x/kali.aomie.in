How to Install gmult on Kali Linux

  • Home
  • Kali Linux
  • gmult

Today in this tutorial, I’ll show you how you can install gmult on Kali Linux with and without using the command line.

Introduction

Installing gmult on Kali Linux is very easy. And with the help of this tutorial, you will be able to easily install, update and uninstall gmult in just a few steps.

In this tutorial I’ll show you four methods to install gmult, the methods are using apt, apt-get, aptitude, and synaptic. All these methods require you to have an active and stable internet connection and sudo permission.

Install gmult Using apt

To install gmult in Kali Linux using apt, you will need to run the apt update command first to get the updated list of packages from the repositories.

Copy the command given below and paste it in the command line, then press enter. If you are using the sudo command for the first time then you might also need to enter your root password.

sudo apt update

After you have updated the package list using apt, you can now install gmult using the apt install command. To do that use the command given below.

sudo apt -y install gmult

The installation of gmult might take a little bit of time depending upon your internet connection and machine configuration.

Install gmult Using apt-get

To install gmult using apt-get in Kali Linux, you will need to first update the package list using the apt-get update command. Copy, paste, and enter the command given below in the command line.

sudo apt-get update

After updating the package lists, use the apt-get install command given below to install gmult.

sudo apt-get -y install gmult

Install gmult Using Synaptic

To install gmult using Synaptic you will need to first install Synaptic, you can do that by following the instructions from here. After installing synaptic follow the steps below to install gmult on your Kali Linux using Synaptic.

You can open synaptic by clicking on the menu icon then searching for synaptic or you can open the command line and enter the command given below –

sudo synaptic

After synaptic opens, click on the search icon and search for gmult. This will display the gmult package in the package window, now to install gmult, click on the checkbox that is on the left side.

Then click on Mark for Installation, this will select the gmult package for installation, now to install it click on the Apply button, you will get a summary window where click on the apply button again to start the download of gmult.

If the download and installation are successful then the checkmark option will turn green.

Install gmult Using aptitude

To install gmult on Kali Linux using aptitude, you will need to first install Aptitude in your Kali Linux. To install Aptitude follow the instructions here. Once you have aptitude installed proceed to the installation of gmult.

Now you will need to first update your package lists from the repositories, use the apt update command for that, after that update the package list of aptitude using the aptitude update command.

sudo apt update sudo aptitude update

After updating the package list of aptitude, install gmult using the command aptitude install gmult.

sudo aptitude -y install gmult

How to Upgrade gmult on Kali Linux

If you want to upgrade the gmult package on your Kali Linux to its latest version then here’s how you can do that using apt, apt-get, aptitude, and synaptic.

Upgrade gmult using Apt & Apt-get

To upgrade gmult to its latest version using apt or apt-get, you will need to open the terminal then run the sudo apt update command to update the package list, after that copy and paste the commands given below to upgrade gmult.

sudo apt install --only-upgrade gmult sudo apt-get install --only-upgrade gmult

Upgrade gmult using Aptitude

To upgrade gmult using aptitude you will need to again run the update command to update the package list, use command sudo aptitude update to do that. After that run the command given below to upgrade gmult.

sudo aptitude -y full-upgrade gmult

Upgrade gmult using Synaptic

To upgrade gmult using synaptic open synaptic using the command sudo synaptic or by going to the menu and searching for synaptic.

In synaptic click on the reload button, this will look for updates. After that you will need to search for gmult, now right-click gmult, and click on the mark for upgrade option.

Now to upgrade gmult just click on the apply button, then click on the apply button again in the summary screen.

How To Uninstall gmult on Kali Linux

If you want to remove and uninstall gmult from your Kali Linux then here’s how you can do that. To remove just gmult, you can use the command that is given below, copy the command, paste it in the command line then press enter.

sudo apt remove gmult

Remove related packages, dependencies & configuration files of gmult

To remove all the related packages, dependencies and configuration files of gmult you can use the command given below.

sudo apt -y purge gmult

Remove gmult using Synaptic Package Manager

You can also uninstall gmult using the synaptic package manager, first open synaptic using the command sudo synaptic or search and open it directly.

After that in synaptic search for gmult, right-click the gmult package and click on the Mark for Complete Removal option. Now to uninstall gmult click on the apply button then in summary click on apply again.

Conclusion

That is how you can install gmult on Kali Linux using the command line or using the synaptic package manager. I also showed you how you can uninstall and upgrade gmult using the command line or using the synaptic package manager.
